* Remove duplicate records
* Correct data formats
* Validate dataset consistency
* Prepare data for further analysis

### Skills Used

* Python
* Pandas
* Data Cleaning
* Data Preparation
* Data Quality Assessment

### Deliverables

* Data Cleaning Notebook
* Cleaned Dataset
