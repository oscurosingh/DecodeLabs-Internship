# Project 3: SQL Data Analysis

## Objective
The objective of this project is to use SQL queries to extract meaningful insights from the provided dataset. The project demonstrates the use of SQL fundamentals such as data selection, filtering, sorting, grouping, and aggregation.

## Dataset
Dataset: Dataset for Data Analytics.xlsx

The dataset contains business transaction records including product details, order information, customer data, payment methods, order status, and revenue-related attributes.

## Tools Used
- Python
- Pandas
- SQLite
- Jupyter Notebook

## Project Tasks

### Task 1: Data Loading
- Load Excel dataset using Pandas
- Create an in-memory SQLite database
- Store the dataset as a SQL table

### Task 2: SELECT Query
- Retrieve specific columns from the dataset

### Task 3: WHERE Clause
- Filter records based on specified conditions

### Task 4: ORDER BY Clause
- Sort records in ascending or descending order

### Task 5: GROUP BY Clause
- Group records based on categorical columns

### Task 6: COUNT Aggregation
- Count records within each group

### Task 7: SUM Aggregation
- Calculate total revenue and other numerical summaries

### Task 8: AVG Aggregation
- Calculate average values across groups

### Task 9: Combined SQL Analysis
- Apply WHERE, GROUP BY, ORDER BY, and aggregation functions together to generate business insights

## Key SQL Concepts Demonstrated
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- COUNT()
- SUM()
- AVG()

## Conclusion
This project demonstrates how SQL can be used to transform raw data into actionable business insights. By combining filtering, grouping, sorting, and aggregation techniques, meaningful patterns and trends can be extracted efficiently from large datasets.
